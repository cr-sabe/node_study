# Node.jsによる気象情報閲覧システムの開発

## 作業手順記録

### 初期構築
1. Express.js による初期化
~~~
$ npm install express-generator request -g
$ express --view=ejs node_study
$ cd node_study
$ npm install
~~~

2. 初期ファイル作成(app.js、index.html)


## 起動方法
1. コマンドから起動
~~~
$ DEBUG=login-app-ejs:* npm start
~~~

2. ブラウザで http://localhost:8080/ にアクセス
